# 🗃️ Analyse RH - Dashboard Excel

## Aperçu du projet

Ce projet consiste à analyser des données RH afin de comprendre les facteurs liés au turnover des employés, d'explorer la répartition des salaires et de l'ancienneté.

L'objectif est de transformer des données RH en **insights exploitables** grâce à un **dashboard interactif construit avec Excel**.

---

## Dashboard

![Dashboard](Projet_1_Analyse_RH/Images/Capture_d'écran.png)

Ce dashboard interactif permet d'analyser :

- Le turnover des employés
- La répartition des salaires
- L'ancienneté dans l'entreprise
- Les départs par département

Les graphiques et KPI se mettent à jour automatiquement grâce aux segments.

---

## Dataset

Source : IBM HR Analytics Dataset

Le dataset contient des informations sur les employés telles que :

- Département
- Poste
- Salaire
- Ancienneté dans l'entreprise
- Départ de l'employé (attrition)

---

## Insights principaux

L'analyse met en évidence plusieurs observations :

- Le département **Ventes présente le turnover le plus élevé**
- Les employés avec **moins de 2 ans d'ancienneté quittent plus souvent l'entreprise**
- Certains départements présentent **des salaires moyens plus faibles**

Ces informations peuvent aider les équipes RH à améliorer la rétention des talents.
